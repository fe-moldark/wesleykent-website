# STL (and MCX) Files for Download
If you want to download these and manipulate them yourself I will share those links down below, nothing is behind a paywall of course. I'll be hosting them here on Github and on Sketchfab since they offer a convenient way to embed the 3d models into web pages.
<br><br>
- The Github directory housing both the STL and MCX formats can be found <a href="https://github.com/fe-moldark/wesleykent-website/tree/gh-pages/assets/3d_files/FinalGameConsole" target="_blank" rel="noopener noreferrer">here</a>
<br>
- My Sketchfab profile can be found <a href="https://sketchfab.com/femoldark" target="_blank" rel="noopener noreferrer">here</a>, which also hosts some other designs I've worked on if you are interested
<br><br><br>

# Preview
I would recommend viewing this page on a larger screen, i.e. not your mobile. The frames should adjust properly now, but you still won't be able to get a good look at the 3d models on a small screen. All of the below files are current as of June '23 and should be the final prints.
<br><br>
_**Note: before printing anything off check the Github link for any "modified..." files in the subdirectories. I will not keep updating the STLs, instead, any updates will be pushed there as individual MCX files.**_
